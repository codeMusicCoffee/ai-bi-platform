## 🚀 如何使用

### 1. AI 开发前的准备

AI 助手在开始任务前，应按以下顺序阅读：

1. **`AGENT.md`** - 了解最高准则和冻结区
2. **`.agent/CHEAT_SHEET.md`** - 快速查找常用规则
3. **对应的指令文件** - 根据任务类型查阅详细规范

### 2. 使用工作流

在对话中直接引用工作流命令：

```
/add-feature 用户权限管理
/fix-bug 产品删除后树未刷新
/add-page 数据统计页面
```

### 3. 查阅架构说明

遇到复杂模块（如产品管理、Sandpack）时，先阅读 `.spec/` 中的对应文档，避免误改核心逻辑。
